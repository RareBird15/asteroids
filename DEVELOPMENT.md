# Development Notes

## Why This File Exists

This project is intentionally minimal and course-scoped. This file captures practical notes for anyone reading or running it later.

## Quick Start

```bash
python main.py
```

## Logging Outputs

The logger writes two newline-delimited JSON files at runtime:

- game_state.jsonl: sampled snapshots of groups/sprites
- game_events.jsonl: event stream (for example: asteroid_shot, asteroid_split, player_hit)
